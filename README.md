# СберАвтоподписка — предсказание конверсии

## 📌 О проекте

Модель машинного обучения для предсказания вероятности целевого действия на сайте **СберАвтоподписка**.

**Целевое действие:** заявка, звонок, открытие диалога, оформление заказа.


## 📊 Результаты модели

| Модель | ROC-AUC |
|--------|---------|
| Logistic Regression | 0.8573 |
| Random Forest | 0.8802 |
| **XGBoost** | **0.8851** ✅ |

**Лучшая модель:** XGBoost  
**Accuracy:** 81%  
**Recall (конверсия):** 86%


## ⚙️ Установка и запуск

### 1. Клонировать репозиторий


git clone 

cd sber_autosubscription_project

### 2. Установить зависимости

pip3 install -r requirements.txt

### 3. Подготовить данные

Положите файлы в папку data:

https://cloud.mail.ru/public/PXoc/hDmWMRLe6

data/

├── ga_sessions.csv

├── ga_sessions.pkl

├── ga_hits.csv

└── ga_hits.pkl

### 4. Запустить API

cd api

python3 app.py


### Интерпретация
| Ответ | Значение |
|--------|---------|
|prediction: 1|	✅ Конверсия вероятна|
|prediction: 0|	❌ Конверсия маловероятна|
|probability: 0.85	|85% вероятность|
